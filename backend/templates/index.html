Características del Dataset

📊 30 Características

Mediciones microscópicas del núcleo celular del tumor.

📈 569 Casos

Instancias de diagnóstico de cáncer de mama reales.

✓ Clasificación Binaria

Maligno (canceroso) o Benigno (no canceroso).

⭐ Modelos Entrenados

Red Neuronal + Regresión Logística con validación cruzada.

🧠 Red Neuronal (MLP)

Modelo con capas ocultas que captura relaciones no lineales complejas entre las características del tumor.

📈 Regresión Logística

Modelo estadístico que proporciona probabilidades bien calibradas e interpretables de malignidad.

⚠️ Descargo de Responsabilidad

IMPORTANTE: Este sistema es para propósitos educativos únicamente. Las predicciones NO deben utilizarse para diagnósticos médicos reales. Siempre consulte con un oncólogo o médico calificado para decisiones médicas.

📚 Información del Proyecto

Dataset: Wisconsin Breast Cancer

  • Fuente: UCI Machine Learning Repository
  • Instancias: 569 casos
  • Características: 30 mediciones
  • Objetivo: Clasificación binaria (Maligno/Benigno)
  • Precisión: >95% en validación

Mediciones Incluidas

  • Radio, Textura, Perímetro
  • Área, Suavidad, Compacidad
  • Concavidad, Puntos cóncavos
  • Simetría, Dimensión fractal
  • Estadísticas: Media, SE, Peor
